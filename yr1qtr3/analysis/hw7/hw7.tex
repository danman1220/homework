\section*{Problem 1}
Suppose $G$ is a locally compact abelian group, with $\mu$ the Haar measure on
$G$. If $G$ is not discrete and $\varepsilon>0$ is fixed, find a compact
neighborhood $K$ of the identity $0$ with $\mu(K)<\varepsilon$.

\begin{proof}
    Sketch: the set $\{0\}$ has measure zero (prove!) and so
    \[
        \mu(\{0\}) = \inf_{0\in U} \mu(U)\to 0
    \]
    so choose $U$ such that $\mu(U)<\varepsilon$ and take $K\subset U$ compact
    by local compactness.
\end{proof}
